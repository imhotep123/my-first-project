# 管理規約改正業務 スクリプト集

マンション管理規約を「令和7年マンション標準管理規約」へ準拠させるための Word 文書（.docx）改正作業支援スクリプト群。

## ファイル構成

| ファイル | 内容 |
|---|---|
| `append_articles.py` | 改正案.docx へ第６章以降を追記するスクリプト（最初のバージョン） |
| `cleanup_and_rebuild.py` | 重複コンテンツを除去して正しい内容を1回だけ再追記するスクリプト |
| `ヒアリングシート回答.md` | 改正指示一覧（改正種別・個別指示・参照条文） |

## 関連スキル

Claude Code で `/kiyaku-kaisei` スキルを呼び出すことで、改正業務をインタラクティブに実行できます。
スキル定義: `../../.claude/commands/kiyaku-kaisei.md`

## 対応種別

- **単棟型**（作業実績あり）
- 団地型（次回以降対応予定）
- 複合用途型（次回以降対応予定）

## 主要ファイルパス（既定）

| 種別 | R7標準規約 |
|---|---|
| 単棟型 | `D:\Claude\☆令和７年標準管理規約（単棟型）電磁的方法あり.docx` |
| 団地型 | `D:\Claude\☆令和７年標準管理規約（団地型）電磁的方法あり.docx` |
| 複合用途型 | `D:\Claude\☆令和７年標準管理規約（複合用途型）電磁的方法あり.docx` |

## 依存ライブラリ
